# Temas para Debian
Fondos para Debian (escritorio, grub, gdm) diseñados por mi y con autoinstalador

## Estructura
- Temas Completos → Incluye temas completos con autoinstalador.
- Fondos de pantalla → Contiene solo fondos de pantallas.
- Fondos GRUB → Contiene solo fondos adaptadas para GRUB
- Fondos GDM → Contiene solo fondos adaptados para GDM

## Instalar Tema Completo
Dentro del apartado de temas completos se incluye un script instalador.

Para instalar el tema solo es necesario navegar hacia el mismo y ejecutar los siguientes comandos.

Dar permisos:
```bash
	chmod +x install.sh
```

Una vez con permisos de ejecución procedemos a ejecutar el script que lo instalará y activará:
```bash
	./install.sh
```
